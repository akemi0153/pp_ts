## Customization

### Brand Colors
The design uses Salinbanwa's brand colors. Edit in `styles.css`:
```css
:root {
    --primary: #2d5f7f;    /* Teal from logo */
    --accent: #a67c6d;     /* Brown from logo */
    --text-dark: #1a1a1a;
    --text-medium: #4a4a4a;
}
```

### Logo
Replace `bg-salinbanwa.png` with your own logo. Recommended size: 400x150px PNG with transparency.

### Content
Edit the HTML files directly to update:
- Company information
- Contact details
- Specific policies and terms
- Footer links
